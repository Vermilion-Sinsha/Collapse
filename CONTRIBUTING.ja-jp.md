# コントリビューションガイドライン
## 最初に
- READMEに別途記載されているものを除いた、Collapseへのコンパイルが必要な全てのサブモジュールをプルするために、`--recurse`パラメーターで`main`ブランチからソースコードをクローンしてください。

    ```git clone https://github.com/neon-nyan/Collapse --recurse```

- 必ず以下のリストに記載されたIDEおよびSDKを使用してください。
- 必ず下記の「機能実装に関する制限」の欄を読んでください。
- READMEに特記されている場合やメンテナーから特別な指示を受けた場合を除いて、プルリクエスト（PR）を送信する時は最初にプルしたブランチと同一のブランチへ送信するようにしてください。
- PRの内容を何でもかんでも説明する必要はありませんが、何が追加/削除されたのか、何を行うためのコードなのかを明確に説明してください。

## 翻訳協力
[Crowdin](https://crowdin.com/project/collapse-launcher)での翻訳の追加/校正にご協力ください！対応言語を増やしてほしい場合は、Githubでissueを作成するか、Crowdinのディスカッションで連絡してください。なるべく早く返信できるように頑張ります。

## 必須ツール
下記のリストは、当プロジェクトに参加するのに必要なツールの一覧です。
1. **Visual Studio 2022 (17.4以降の全エディション)**
   - .NET デスクトップ開発コンポーネントを選択してください
2. Visual Studioインストーラーから入手した**Windows SDK (10.0.19043.0または10.0.22000.0のみ)**
3. .NET Core 7 SDK: [**(7.0.100以降)**](https://dotnet.microsoft.com/en-us/download/dotnet/7.0)
4. WinUI 3: [**(WindowsAppSDK 1.3.0-230331000安定版ランタイム)**](https://aka.ms/windowsappsdk/1.3/1.3.230331000/windowsappruntimeinstall-x64.exe)

> **注意**:
> 
> 2022年11月13日から、`WinUI 3 1.3 Stable`の最小システム要件が更新されたため、Visual Studio 2022が<b>必ず</b>インストールされている必要があります。
> 
> より古いバージョンのVisual Studio（VS2019など）も使用可能ですが、ビルド前に*NuGet*から**WindowsAppSDK**を**WindowsAppSDK 1.1.5**または**WindowsAppSDK 1.2-preview2**にダウングレードする必要があるため、非推奨です。これによりアプリケーションが破壊されるリスクが高まるため、この方法へのサポートは提供されません。**初心者にはオススメしません。**

## 機能実装に関する制限
当ソフトウェアは完全オープンソースかつHoYoverse、Cognosphere、その他関連団体とは一切関係していませんが、Collapseを開発する際はそれらの利用規約および行動規範を遵守してください。これは、Collapseに決して実装されない機能があることを意味します。以下にその一部を列記します。これらの機能追加に関するissue、PRは全てクローズします。
- ログインや支払い認証エンドポイントなどのmiHoYo SDKまたはAPIへ、何らかの方法で干渉するもの
- DLLインジェクションやメモリ読み取り/書き込みなど、ゲームプロセス（およびサブプロセス）に直接介入するもの
- ゲームファイルやリソースの直接的な書き換えによって、当ソフトウェアを利用するユーザーにゲーム内での不公平な利益を与えるもの

### PRの送信を推奨する機能の例
- ファンが制作したデータベース
- HoYoverse公式/公認ツール
- などなど
  
### 絶対に実装しない機能の例
- アカウント切り替え
- FPS上限解除（上記ルールに違反するものに限る）
- あらゆるチート、エクスプロイト
- などなど
